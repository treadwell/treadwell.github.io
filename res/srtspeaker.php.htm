Subtitles to speech converter

The tool is designed for converting subtitles in SubRip format (SRT) to speech in real time using TTS chrome engine. It helps also to edit and debug subtitles. When used with the SpeechPad transcription module, it allows to translate a sound track of the video to the foreign language. Detailed description of using the tool is in the SpeechPad.pw blog. The utility works only in Google Chrome browser. We worked out also TTS Picker utility, that read the sentences from list.
